# linux-rebuild-manager

linux-rebuild-manager helps migrate installed applications and user
configurations to rebuild a familiar Linux working environment on
a new system.

## Features
- Captures installed apps (APT, Snap, Flatpak)
- Saves user-level configuration data
- Assists in recreating the same environment on another machine

## Non-goals
- No disk imaging
- No filesystem snapshots
- No OS backup or restore

## Supported systems
- Ubuntu / Ubuntu-based (full)
- Debian-based (partial)
- Other Linux (Flatpak + configs only)
